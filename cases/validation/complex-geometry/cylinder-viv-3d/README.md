# VIV Cylinder (3D)

## Physics

3D vortex-induced vibration. The spanwise correlation of vortex shedding affects the cylinder response amplitude. Key phenomena:
- Oblique vs parallel vortex shedding
- Spanwise de-correlation at low m*
- Transition between shedding modes along span

## Reference Data

- integral.csv -- 2D VIV parameters
- spanwise_correlation.csv -- lift correlation vs spanwise separation
