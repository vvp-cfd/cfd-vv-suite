# Backward-Facing Step (3D)

## Physics

3D laminar flow over a backward-facing step with side walls. The end-wall boundary layers create spanwise non-uniformity in the recirculation zone. For spanwise aspect ratio Ar = W/h >= 4, the midplane reattachment length approaches the 2D Armaly result.

## Differences from 2D

- Side wall boundary layers reduce effective channel width
- Reattachment length varies in z: shorter near side walls, longer at midplane
- Secondary flow (u_z) near corners

## Reference Data

2D Armaly data at midplane (z=W/2) plus integral quantities.
