# Turbulent Channel Flow (Re_tau=1000)

## Physics

High-Re turbulent channel. Extended log-law region (y+ from 30 to ~300). Tests RANS model accuracy at moderate Reynolds numbers.

## Expected Quantities

| Quantity | Value |
|----------|-------|
| Re_tau | 1000 |
| Re_bulk | 24000 |
| Cf | 0.00357 |
| Ub+ | 22.1 |
