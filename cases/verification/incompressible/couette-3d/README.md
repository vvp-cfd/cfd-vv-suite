# Plane Couette Flow (3D)

## Physics

Shear-driven flow in a 3D domain. 2D physics (no z-dependence), tests spanwise uniformity.

## Analytical Solution

u(y) = U0 * y/H, v = w = 0, p = const

## Boundary Conditions

| Boundary | Condition |
|----------|-----------|
| Bottom (y=0) | No-slip |
| Top (y=H) | Moving: u=U0 |
| Inlet/Outlet | Profile inlet or periodic |
| Spanwise (z) | Periodic or symmetry |
