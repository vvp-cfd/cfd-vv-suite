# Taylor-Couette Flow (3D)

## Physics

3D flow between rotating cylinders with end walls. Above critical Taylor number Ta_c=1708, toroidal Taylor vortices appear, creating axial periodicity.

## Regimes

| Ta | Flow |
|-----|------|
| < 1708 | Steady axisymmetric Couette |
| 1708-3000 | Steady Taylor vortices |
| > 3000 | Wavy Taylor vortices (time-dependent) |
| > 10000 | Turbulent Taylor vortices |

## Reference Data

Axisymmetric solution u_theta(r) for Ta < Ta_crit. For Ta > Ta_crit, the 3D flow deviates from this.
