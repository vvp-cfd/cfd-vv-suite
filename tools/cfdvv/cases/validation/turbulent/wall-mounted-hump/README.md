# NASA Wall-Mounted Hump

## Physics

Turbulent flow over a Glauert-Goldschmied wall-mounted hump. A separation bubble forms on the aft portion of the hump (x/c ~ 0.65 to 1.11). This is Case 3 from the NASA CFDVAL2004 Workshop.

## Geometry

Hump mounted on a splitter plate in a wind tunnel. Chord-based Reynolds number Re_c = 936000, M = 0.1 (effectively incompressible).

## Flow Features

- Favorable pressure gradient on forebody (x/c < 0.5)
- Adverse pressure gradient on aft body causes separation
- Reattachment around x/c = 1.11
- Cp distribution sensitive to turbulence model

## Reference Data

- cp_surface.csv -- Cp(x/c) on hump surface and downstream wall
- integral.csv -- separation/reattachment points
